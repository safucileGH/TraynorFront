# 🏋️ Traynor — Fitness Professional Marketplace

Traynor is a web platform that connects fitness professionals 
(boxing, yoga, crossfit, bodybuilding, and more) with clients 
looking for personalized training.

## ✨ Features

- 🔍 Search trainers by name or training type
- 👤 Trainer profile cards with full details
- 🔐 Admin login panel to validate real professionals
- ✅ Trainer approval workflow before appearing in search
- 📱 Responsive design for mobile and desktop

## 🛠️ Tech Stack

**Frontend**
- React.js
- HTML5 / CSS3
- JavaScript (ES6+)

**Backend** → [TraynorSiEs Repo](https://github.com/safucileGH/TraynorSiEs)
- Node.js + Express.js
- MySQL (relational database)
- JWT (JSON Web Tokens) for admin authentication

## ⚙️ Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn
- Backend server running (see TraynorSiEs repo)

### Installation

```bash
git clone https://github.com/safucileGH/TraynorFront.git
cd TraynorFront
npm install
npm start
```

The app runs on `http://localhost:3000`
